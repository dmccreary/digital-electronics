# References: Number Systems and Binary Arithmetic

1. [Binary number](https://en.wikipedia.org/wiki/Binary_number) - Wikipedia - Comprehensive overview of binary number systems, positional notation, and arithmetic operations.

2. [Two's complement](https://en.wikipedia.org/wiki/Two%27s_complement) - Wikipedia - Detailed explanation of two's complement representation and why it simplifies hardware design.

3. [Gray Code](https://en.wikipedia.org/wiki/Gray_code) - Wikipedia - Comprehensive reference on Gray code history, properties, and applications in rotary encoders and error correction.

4. Digital Design (6th Edition) - M. Morris Mano and Michael Ciletti - Pearson - The standard undergraduate textbook covering number systems, binary arithmetic, and two's complement representation in Chapters 1-2.

5. [Binary Arithmetic](https://www.allaboutcircuits.com/textbook/digital/chpt-2/binary-arithmetic/) - All About Circuits - A comprehensive free online textbook chapter covering binary addition, subtraction, and signed number representations with worked examples.
