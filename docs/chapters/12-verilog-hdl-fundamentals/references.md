# References: Verilog HDL Fundamentals

1. [Verilog](https://en.wikipedia.org/wiki/Verilog) - Wikipedia - Comprehensive overview of Verilog HDL history, syntax, and applications in digital design.

2. [Hardware description language](https://en.wikipedia.org/wiki/Hardware_description_language) - Wikipedia - General introduction to HDLs and their role in digital system design.

3. [Verilog Tutorial](https://www.chipverify.com/verilog/verilog-tutorial) - ChipVerify - Comprehensive Verilog tutorial covering modules, ports, data types, and assignments with examples.

4. [HDLBits: Verilog Practice](https://hdlbits.01xz.net/wiki/Main_Page) - HDLBits - Interactive Verilog exercises with instant feedback, excellent for learning syntax and concepts.

5. IEEE Standard 1364-2005 - IEEE - The official Verilog language reference manual (IEEE Std 1364-2005).
