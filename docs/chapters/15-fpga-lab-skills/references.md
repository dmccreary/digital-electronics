# References: FPGA Lab Skills

1. [Field-programmable gate array](https://en.wikipedia.org/wiki/Field-programmable_gate_array) - Wikipedia - Comprehensive overview of FPGA architecture, programming, and applications.

2. [Lookup table](https://en.wikipedia.org/wiki/Lookup_table#Hardware_LUTs) - Wikipedia - Explanation of LUT-based logic implementation in FPGAs.

3. [Digilent FPGA Tutorials](https://digilent.com/reference/learn/programmable-logic/tutorials/start) - Digilent Reference - Hands-on tutorials for FPGA development with Basys 3 and Nexys boards.

4. [Nandland FPGA Fundamentals](https://nandland.com/fpga-101/) - Nandland - Beginner-friendly FPGA tutorials covering architecture, implementation flow, and debugging.

5. [HDLBits: Verilog Practice](https://hdlbits.01xz.net/wiki/Main_Page) - HDLBits - Interactive Verilog exercises suitable for FPGA implementation practice.
