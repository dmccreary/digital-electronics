# References: FSM Design and Applications

1. [Finite-state machine](https://en.wikipedia.org/wiki/Finite-state_machine) - Wikipedia - Comprehensive overview of FSM theory with examples of sequence detectors and controllers.

2. [Sequence detector](https://en.wikipedia.org/wiki/Finite-state_machine#Example:_sequence_detector) - Wikipedia - Example of FSM application in pattern recognition.

3. Digital Design (6th Edition) - M. Morris Mano and Michael Ciletti - Pearson - Chapter 7 includes FSM design examples including sequence detectors and controller applications.

4. [Finite State Machines in Embedded Systems](https://users.ece.utexas.edu/~valvano/Volume1/E-Book/C10_FiniteStateMachines.htm) - UT Austin - Embedded systems perspective on FSM design with traffic light controller example.

5. [Verilog Code for Moore FSM Sequence Detector](https://www.fpga4student.com/2017/09/verilog-code-for-moore-fsm-sequence-detector.html) - FPGA4student - Complete tutorial on implementing sequence detectors with Verilog code and simulation.
